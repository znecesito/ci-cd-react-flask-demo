# React + Flask CI/CD Demo

This repo contains a React frontend and Flask backend with CI/CD via GitHub Actions.

## How to Run Locally
```bash
# Backend
cd backend
pip install -r requirements.txt
python app.py

# Frontend
cd frontend
npm install
npm run dev
```
